🚀 クイックスタート

  1. リポジトリをクローン

    git clone https://github.com/maosasa/event-board.git

  2. uv をインストール

    curl -LsSf https://astral.sh/uv/install.sh | sh

  3. 環境変数を設定

    cp .env.example .env

    PostgreSQL 接続情報を .env に設定

  4. 依存関係をインストール

    uv sync

  5. アプリケーションを実行

    uv run python app.py

詳細は GitHub の README を参照してください
